Step 4= Embedding layer - every ID ( number here ) is converted into a Vector of size - "Embedding dimension" , normally 256 or 1024
Step 5= GRU layer - reads character one at a time and maintains hidden state(memory) - ( remembers entire sequence context )
Step 6= Dense Layer - converts the 1024 features into outputs in the same vector feature format
